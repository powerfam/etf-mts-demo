## 왜 이 서비스가 필요한가?

### 현재 ETF 투자의 문제점

```
┌─────────────────────────────────────────────────────┐
│                   개인 투자자의 고민                  │
├─────────────────────────────────────────────────────┤
│  "ETF가 좋다는데... 뭘 사야 하지?"                    │
│  "이 ETF 믿어도 되나? 수수료는?"                      │
│  "내 연금계좌에 이거 담아도 되나?"                    │
│  "비슷한 ETF가 너무 많아서 뭐가 뭔지..."             │
└─────────────────────────────────────────────────────┘
```

### 기존 MTS의 한계

| 문제 | 설명 |
|------|------|
| **정보 파편화** | 건전성 지표가 여러 화면에 흩어져 있음 |
| **검증 어려움** | 괴리율, 스프레드 등 확인하려면 전문 지식 필요 |
| **계좌 구분 부재** | 연금계좌 부적합 상품 경고 없음 |
| **비교 불편** | ETF 간 비교하려면 수동으로 데이터 수집 |

## 고객이 얻는 가치

### 1. 한눈에 보는 ETF 건전성

> **"이 ETF, 믿어도 되나요?"에 대한 명확한 답**

```
┌─────────────────────────────────────────┐
│         건전성 점수: 87점                │
│         ████████████████░░░░            │
├─────────────────────────────────────────┤
│  ✓ 괴리율 0.12%    → 양호               │
│  ✓ 스프레드 0.08%  → 우수               │
│  ✓ TER 0.07%       → 최저 수준          │
│  ✓ 거래대금 892억  → 풍부               │
└─────────────────────────────────────────┘
```

**고객 혜택:**
- 복잡한 지표를 **0~100점 단일 점수**로 이해
- 투자 전 **리스크 사전 인지**
- 전문가 수준의 검증을 **클릭 한 번**으로

---

### 2. 계좌 맞춤형 안전장치

> **"내 연금계좌에 이거 담아도 되나요?"**

| 계좌 | 자동 필터링 | 세금 안내 |
|------|------------|----------|
| 연금 | 레버리지/인버스 숨김 | 5.5% 세율 표시 |
| ISA | 부적합 상품 경고 | 9.9% 세율 + 비과세 한도 |
| 일반 | 제한 없음 | 15.4% 세율 표시 |

**고객 혜택:**
- **실수로 부적합 상품 매수 방지**
- 계좌별 **예상 세금 자동 계산**
- 연금계좌 최적화 상품 **자동 추천**

---

### 3. 쉬운 비교, 빠른 결정

> **"KODEX 200이랑 TIGER 200 뭐가 달라요?"**

```
┌─────────────────────────────────────────────────────┐
│              ETF 비교 (최대 3개)                     │
├─────────────┬─────────────┬─────────────────────────┤
│   KODEX 200 │  TIGER 200  │    차이점               │
├─────────────┼─────────────┼─────────────────────────┤
│ TER: 0.05%  │ TER: 0.05%  │  동일                   │
│ 괴리율: 0.1%│ 괴리율: 0.08%│ ← TIGER 우수           │
│ 거래대금: 5천억│거래대금: 3천억│ ← KODEX 우수        │
│ 건전성: 92  │ 건전성: 90   │  비슷                  │
└─────────────┴─────────────┴─────────────────────────┘
```

**고객 혜택:**
- 유사 ETF **핵심 차이점 즉시 파악**
- 감이 아닌 **데이터 기반 선택**
- 비교 시간 **90% 단축**

---

### 4. 투자 교육 통합

> **"ETF가 뭔지부터 알고 싶어요"**

- **ETF 101**: 기초 개념부터 고급 전략까지
- **용어사전**: NAV, TER, AP, LP 등 쉬운 설명
- **AI 챗봇**: 궁금한 점 즉시 질문
- **리서치 PDF**: 주간 ETF 분석 리포트

**고객 혜택:**
- **학습과 투자를 한 앱에서**
- 모르는 용어 **클릭 한 번**으로 설명
- 투자 자신감 향상

## 회사가 얻는 기대효과

### 1. 비즈니스 성과

| 지표 | 기대효과 | 근거 |
|------|----------|------|
| **ETF 거래대금** | +15~25% 증가 | 탐색→매수 전환율 향상 |
| **연금계좌 ETF 비중** | +20% 증가 | 적합 상품 추천 기능 |
| **신규 고객 유입** | +10% 증가 | 차별화된 UX로 입소문 |
| **고객 이탈률** | -5% 감소 | 투자 경험 만족도 향상 |

---

### 2. 고객 관계 강화

```
기존 MTS                    ETF 전문 MTS
────────                    ────────────
[단순 거래 도구]      →     [투자 파트너]

"주문만 하는 앱"      →     "검증하고 배우는 앱"

낮은 체류 시간        →     높은 engagement

가격 경쟁             →     가치 경쟁
```

---

### 3. 리스크 관리

| 영역 | 효과 |
|------|------|
| **투자자 보호** | 부적합 상품 투자 사전 방지 → 민원 감소 |
| **정보 투명성** | 건전성 지표 공개 → 신뢰도 향상 |
| **규제 대응** | 연금계좌 적합성 자동 필터 → 컴플라이언스 강화 |

## 핵심 화면 미리보기

### 홈 화면
- 포트폴리오 요약 (계좌별)
- 실시간 인기 ETF (물결 애니메이션)
- 테마별 수익률 히트맵
- 시장 현황 대시보드

### 탐색 화면
- 3가지 모드: 탐색 / 검증 / 주문
- 스마트 필터링 (연금적합, 테마별)
- 정렬 옵션 (건전성순, 저비용순)

### 상세 화면
- 건전성 점수 + 상세 지표
- 가격 차트 + 기술적 정보
- 원클릭 매수/매도
