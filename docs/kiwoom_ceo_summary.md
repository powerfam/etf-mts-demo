## 3️⃣ 2026년 신년사 핵심 (2026.01.02)

| 구분 | 내용 |
|------|------|
| **시장 진단** | 금융환경 그 어느 때보다 빠르게 변화, 기술 진보로 경쟁의 축 변화 |
| **포지셔닝** | 2026년 = 다음 단계 도약을 위한 중요한 전환점 |
| **중점과제 1** | **IT 경쟁력 확보** - AI, 데이터, 시스템안정성, 정보보안, 서비스 아키텍처 전방위 강화 |
| **중점과제 2** | **변화의 속도 제고** - 의사결정·서비스 출시·실패 수정 속도 높이기 |
| **중점과제 3** | **자산관리 표준 도약** - 주식거래 표준 → 자산관리 표준으로 진화, 퇴직연금 확대 |
| **중점과제 4** | **수익구조 다각화** - 발행어음·퇴직연금 본원적 경쟁력 확보 |
| **중점과제 5** | **사회적 책무** - 금융소비자 보호, 모험자본 공급, 생산적 금융 |
| **핵심 발언** | "키움의 시작과 성장은 IT 경쟁력과 함께, 지점영업인력이 아닌 시스템으로 신뢰" |
| **서비스 언급** | AI PB 서비스 고도화 → 앱에서 자산관리 편의성 증대 |
