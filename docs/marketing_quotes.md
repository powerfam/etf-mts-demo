# 마케팅 1줄 서비스 정의 — Claude Code 작업 가이드

## 작업 컨텍스트

키움증권 ETF 대메뉴(주문·탐색·비교 총망라) 프로젝트의
C레벨 보고용 1줄 서비스 정의(포지셔닝 문장)를 작성한다.

---

## 주문 화면 예시 
- (우리 웹사이트에서 다루지않은 다른 사람이 작업한 "주문"화면 정의)

#### 확정된 방향
- **최종 문장(예시)**: "정했다면, 여기에서 끝난다."
- **회사 전략축과의 연결**: 키움증권 2026 중점과제 
  "주식거래 표준 → 자산관리 표준으로 진화"에 직접 대응
- 이건 고객의 동선 데이터를 파악해서 one page 주문화면을 최적화하는 작업에 해당하는 문구임.

---

#### 타겟 고객 정의
- 주문 전 확인해야 할 정보가 여러 화면에 흩어져 있음
  (잔고, 시장지수, 선물가격 등)
- "왜 여기저기 돌아다니야 하지?"라는 피로감을 느끼는 사람

---

## Copywriting 원칙

1. Don't write to impress → 기능 설명이 아니라 고객이 느끼는 결과를 표현
2. Write so a 12-year-old gets it → 복잡한 금융 용어 제외
3. Clarity always beats clever → 영리한 표현보다 명확한 표현 우선

---

## 작업 요구사항

1. "정했다면, 여기에서 끝난다."를 기본축으로 유지
2. C레벨 보고서 slide 제목 자리에 붙을 수 있는 무게와 간결함 유지
3. 한국어 자연스러운 리듬 (번역체 냄새 없음)
4. 아래 두 가지를 참고로 활용:
   - 레퍼랑스: "Hear what matters. Mute the mess." (행동 + 제거의 대비 구조)
   - 회사 전략문장: "주식거래 표준 → 자산관리 표준으로 진화"

---

## 산출물 형태

총 3가지 후보를 작성하고, 각각에 대해 아래를 붙어라:
- 강점 1줄
- 약점 1줄  
- 권장 사용 컨텍스트 1줄 (예: slide 제목 / 본문 / 소구포인트)
