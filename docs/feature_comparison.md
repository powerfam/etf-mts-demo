# ETF MTS 화면 기획 비교 분석

## 기획 개요

| 구분 | 원안 (빠른 검색 중심) | 데모 (분석·교육 통합형) |
|:---:|:---|:---|
| **핵심 컨셉** | 주제별 버튼 → ETF 랭킹 조회 | 시각화 + 건전성 검증 + 비교 분석 |

| **목표 사용자** | 빠른 매매 원하는 투자자 | 정보 기반 의사결정 투자자 |

## 1. 원안 기능 구성

**[ETF 빠른 검색]** 영역에 주제별 버튼 배치

| 주제 버튼 | 설명 |
|:---|:---|
| 요즘 뜨는 테마 ETF | 기본 선택값 |
| 자금 유입이 많은 | 최근 순자산 증가 기준 |
| 고객이 어제 많이 산 | 전일 매수 거래량 기준 |
| 지금 사면 배당받는 | 배당 기준일 임박 종목 |
| 운용보수 낮은 | TER 낮은 순 정렬 |

→ 버튼 클릭 시 해당 조건의 **ETF 랭킹 리스트** 출력

## 2. 데모 화면 차별화 포인트

### A. 탐색 화면 구조적 차이 ⭐

#### 원안의 한계: 1차원 정렬

```
[원안 구조]
┌─────────────────────────────────────┐
│  정렬 기준: 수익률순 (기본)          │
│  ┌─────────────────────────────────┐│
│  │ 정렬 옵션 모달                  ││
│  │ ○ 자금유입순  ○ 거래량순       ││
│  │ ○ 배당수익률순  ○ TER순        ││
│  └─────────────────────────────────┘│
├─────────────────────────────────────┤
│  ETF A    +3.2%                     │  ← 수익률만 표시
│  ETF B    +2.8%                     │  ← TER, 괴리율 등 안 보임
│  ETF C    +2.1%                     │
└─────────────────────────────────────┘

→ 정렬 기준 변경 시 해당 값만 표시, 나머지 지표 사라짐
→ "TER순" 선택하면 수익률 정보 확인 불가
```

#### 데모의 해결: 다차원 동시 조회

```
[데모 구조 - 검증 모드]
┌─────────────────────────────────────────────────────┐
│  [탐색]  [검증]  [주문]     ← 3가지 뷰 모드 전환    │
├─────────────────────────────────────────────────────┤
│  정렬: 건전성순 | 저비용순 | 유동성순 | 수익률순    │
├──────┬────────┬────────┬────────┬────────┬─────────┤
│ 종목 │  TER   │ 괴리율 │스프레드│ 거래량 │ 건전성  │
├──────┼────────┼────────┼────────┼────────┼─────────┤
│ A    │ 0.07%  │ 0.02%  │ 0.05%  │ 850억  │  92점   │
│ B    │ 0.15%  │ 0.08%  │ 0.12%  │ 320억  │  78점   │
│ C    │ 0.45%  │ 0.25%  │ 0.18%  │  45억  │  61점   │
└──────┴────────┴────────┴────────┴────────┴─────────┘

→ 정렬 기준 변경해도 모든 지표 동시 표시
→ 복수 조건을 한눈에 비교하며 의사결정 가능
```

| 비교 항목 | 원안 | 데모 |
|:---|:---|:---|
| **정렬 차원** | 단일 기준 | 복수 기준 동시 |
| **지표 표시** | 선택 기준값만 | 전체 지표 테이블 |
| **정렬 변경 시** | 다른 정보 사라짐 | 모든 정보 유지 |
| **뷰 모드** | 단일 리스트 | 탐색 / 검증 / 주문 3종 |
| **비교 용이성** | 기준 바꿔가며 재조회 | 한 화면에서 완결 |

> **가치**: 정렬 바꿀 때마다 정보 손실 → **모든 지표 동시 비교**로 다각도 검증

## 4. 결론

| 원안 강점 | 데모 강점 |
|:---|:---|
| 직관적 진입, 빠른 조회 | 깊이 있는 분석, 투자자 보호 |

**데모의 핵심 차별 가치**:
> "무엇이 인기 있는가" → **"왜 이 ETF를 선택해야 하는가"** 로 관점 전환

원안의 빠른 검색 기능을 **히트맵 테마 클릭 → TOP10 모달**로 흡수하면서,
건전성 검증·비교 분석·교육 콘텐츠를 추가하여 **정보 기반 투자 의사결정**을 지원하는 구조.
